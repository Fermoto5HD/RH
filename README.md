# Factor RH Creativo
Este repositorio muestra la estructura y archivos del sitio web de la peluquería Factor RH Creativo en su 4ta versión. 
El sitio se creó totalmente desde cero, implementando Bootstrap 4.1, Bootstrap Native y Webpack. 

## Ejecución 
* Instalar los módulos con `npm install`.
* Iniciar página con `node index.js`. 

## Changelog 
* El Changelog del sitio se desarrollará dentro de un archivo `CHANGELOG.md`. 

## Reutilización de archivos 
Sentite libre en utilizar los distintos archivos de este repositorio para lo que se te ocurra o para algún proyecto que tengas!
